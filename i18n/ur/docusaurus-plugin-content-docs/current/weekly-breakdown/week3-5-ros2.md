# ہفتہ 3-5: ROS 2 کی بنیادی باتیں

## شامل موضوعات

- ROS 2 آرکیٹیکچر اور بنیادی تصورات
- نوڈس، ٹاپکس، سروسز اور ایکشنز
- Python کے ساتھ ROS 2 پیکیجز بنانا
- لانچ فائلیں اور پیرامیٹر مینجمنٹ

## سیکھنے کے مقاصد

ہفتہ 5 کے آخر تک، طلباء قابل ہوں گے:
- شروع سے ROS 2 پیکیجز بنانا
- Python میں پبلشرز اور سبسکرائبرز لکھنا
- سروس سرورز اور کلائنٹس کو لاگو کرنا
- متعدد نوڈس شروع کرنے کے لیے لانچ فائلیں استعمال کرنا
- مختلف کنفیگریشنز کے لیے پیرامیٹرز کا انتظام کرنا
