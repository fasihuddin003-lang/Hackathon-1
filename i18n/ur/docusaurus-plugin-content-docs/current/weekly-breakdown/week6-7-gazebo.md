# ہفتہ 6-7: Gazebo کے ساتھ روبوٹ سمیلیشن

## شامل موضوعات

- Gazebo سمیلیشن ماحول کی سیٹ اپ
- URDF اور SDF روبوٹ ڈیسکرپشن فارمیٹس
- فزکس سمیلیشن اور سینسر سمیلیشن
- روبوٹ وزیولائزیشن کے لیے Unity کا تعارف

## سیکھنے کے مقاصد

ہفتہ 7 کے آخر تک، طلباء قابل ہوں گے:
- Gazebo سمیلیشن ماحول سیٹ اپ کرنا
- روبوٹس کے لیے URDF اور SDF ماڈلز بنانا
- سینسرز (LiDAR، کیمرے، IMUs) سمیلیٹ کرنا
- Unity میں روبوٹس کو وزیولائز کرنا
- کنٹرول کے لیے Gazebo کو ROS 2 کے ساتھ انٹیگریٹ کرنا
