# ہارڈویئر کی ضروریات

## جائزہ

یہ کورس تین بھاری کمپیوٹیشنل لوڈز کے سنگم پر واقع ہے:
- فزکس سمیلیشن (Isaac Sim/Gazebo)
- بصری پرسیپشن (SLAM/کمپیوٹر وژن)
- جنریٹو AI (LLMs/VLA)

## 1. "ڈیجیٹل ٹوئن" ورک سٹیشن

| کمپوننٹ | ضرورت | نوٹس |
|-----------|-------------|-------|
| GPU | RTX 4070 Ti (12GB VRAM) یا اس سے زیادہ | RTX 3090/4090 (24GB) بہترین |
| CPU | Intel Core i7 (13th Gen+) / AMD Ryzen 9 | فزکس کیلکولیشنز CPU-intensive ہیں |
| RAM | 64 GB DDR5 (32 GB کم از کم) | پیچیدہ سینز کو زیادہ میموری درکار ہے |
| OS | Ubuntu 22.04 LTS | ROS 2 Linux کے لیے native ہے |

## 2. "فزیکل AI" ایج کٹ

| کمپوننٹ | ماڈل | مقصد |
|-----------|-------|---------|
| دماغ | NVIDIA Jetson Orin Nano (8GB) / Orin NX (16GB) | ایمبوڈیڈ AI کے لیے انڈسٹری معیار |
| آنکھیں | Intel RealSense D435i یا D455 | VSLAM کے لیے RGB اور ڈیپتھ ڈیٹا |
| بیلنس | Generic USB IMU (BNO055) | انرشیل میژرمنٹ |
| آواز | USB مائیکروفون/اسپیکر اری (ReSpeaker) | وائس ٹو ایکشن انٹیگریشن |

## 3. روبوٹ آپشنز

### آپشن A: پراکسی اپروچ (بجٹ)
- **Unitree Go2 Edu** (~$1,800-$3,000)
- بہترین ROS 2 سپورٹ کے ساتھ کواڈروپیڈ پلیٹ فارم

### آپشن B: منی ایچر ہیومنائیڈ
- **Unitree G1** (~$16k) یا **Robotis OP3** (~$12k)
- **بجٹ**: Hiwonder TonyPi Pro (~$600)

### آپشن C: پریمیم لیب
- **Unitree G1 ہیومنائیڈ** sim-to-real ڈپلائمنٹ کے لیے

## آرکیٹیکچر کا خلاصہ

| کمپوننٹ | ہارڈویئر | فنکشن |
|-----------|----------|----------|
| سم رگ | RTX 4080 + Ubuntu 22.04 والا PC | Isaac Sim, Gazebo, Unity, LLM ٹریننگ |
| ایج برین | Jetson Orin Nano | انفرنس اسٹیک ڈپلائمنٹ |
| سینسرز | RealSense کیمرہ + LiDAR | حقیقی دنیا کا ڈیٹا انپٹ |
| ایکچیویٹر | Unitree Go2 یا G1 (مشترکہ) | موٹر کمانڈز پر عملدرآمد |
