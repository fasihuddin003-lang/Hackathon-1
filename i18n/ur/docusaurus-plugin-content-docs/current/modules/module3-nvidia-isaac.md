# ماڈیول 3: AI-روبوٹ برین (NVIDIA Isaac™)

**فوکس: جدید پرسیپشن اور ٹریننگ۔**

## جائزہ

NVIDIA Isaac AI-powered روبوٹکس کے لیے ایک پلیٹ فارم فراہم کرتا ہے جو سمیلیشن، پرسیپشن اور ری انفورسمنٹ لرننگ کو ملا کر ذہین روبوٹک سسٹمز بناتا ہے۔

## Isaac Sim

Isaac Sim NVIDIA Omniverse پر بنا ایک فوٹو ریئلسٹک سمیلیشن ماحول ہے:

- **فزکس سمیلیشن**: درست rigid body ڈائنامکس
- **سنتھیٹک ڈیٹا جنریشن**: لیبلڈ ٹریننگ ڈیٹا تیار کرنا
- **ڈومین رینڈمائزیشن**: ماڈل کی مضبوطی بڑھانے کے لیے ماحول کو مختلف کرنا
- **Ray Tracing**: RTX-powered فوٹو ریئلسٹک رینڈرنگ

```python
from isaacsim import SimulationApp
app = SimulationApp({"headless": False})

from isaacsim.core.api import World
world = World()
robot = world.add_robot("humanoid")
world.reset()
```

## Isaac ROS

Isaac ROS ہارڈویئر-ایکسلریٹڈ ROS 2 پیکجز فراہم کرتا ہے:

### ہارڈویئر-ایکسلریٹڈ VSLAM (بصری SLAM)

VSLAM روبوٹ کو اپنے ماحول کا نقشہ بنانے اور کیمرہ ڈیٹا کا استعمال کرتے ہوئے اپنی پوزیشن ٹریک کرنے کے قابل بناتا ہے:

```bash
# Isaac ROS VSLAM لانچ کریں
ros2 launch isaac_ros_vslam isaac_ros_vslam.launch.py
```

### نیویگیشن (Nav2)

Navigation2 (Nav2) پاتھ پلاننگ اور رکاوٹ سے بچاؤ کا کام کرتا ہے:

- **گلوبل پلانر**: شروع سے مقصد تک راستہ پلان کرتا ہے
- **لوکل پلانر**: رکاوٹوں سے بچنے کے لیے ریئل ٹائم میں راستہ ایڈجسٹ کرتا ہے
- **ریکوری رویے**: پھنس جانے کی صورت میں ہینڈلنگ

```bash
# Nav2 کو ہیومنائیڈ روبوٹ کے ساتھ لانچ کریں
ros2 launch nav2_bringup navigation_launch.py
```

## دوپائی حرکت کے لیے پاتھ پلاننگ

دوپائی لوکوموشن کے لیے خصوصی پلاننگ درکار ہے:

- **فٹ سٹیپ پلاننگ**: ہر پاؤں کہاں رکھنا ہے
- **بیلنس کنٹرول**: ماس کے مرکز کو برقرار رکھنا
- **ٹریجیکٹری آپٹیمائزیشن**: ہموار، توانائی سے بھرپور حرکت
