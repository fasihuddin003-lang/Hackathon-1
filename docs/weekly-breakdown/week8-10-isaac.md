# Weeks 8-10: NVIDIA Isaac Platform

## Topics Covered

- NVIDIA Isaac SDK and Isaac Sim
- AI-powered perception and manipulation
- Reinforcement learning for robot control
- Sim-to-real transfer techniques

## Learning Objectives

By the end of Week 10, students should be able to:
- Set up and navigate NVIDIA Isaac Sim
- Implement AI-powered perception pipelines
- Train controllers using reinforcement learning
- Apply sim-to-real transfer for real-world deployment
- Use Isaac ROS for hardware-accelerated VSLAM
