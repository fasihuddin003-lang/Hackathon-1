# Weeks 6-7: Robot Simulation with Gazebo

## Topics Covered

- Gazebo simulation environment setup
- URDF and SDF robot description formats
- Physics simulation and sensor simulation
- Introduction to Unity for robot visualization

## Learning Objectives

By the end of Week 7, students should be able to:
- Set up a Gazebo simulation environment
- Create URDF and SDF models for robots
- Simulate sensors (LiDAR, cameras, IMUs)
- Visualize robots in Unity
- Integrate Gazebo with ROS 2 for control
