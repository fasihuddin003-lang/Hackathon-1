# Weeks 3-5: ROS 2 Fundamentals

## Topics Covered

- ROS 2 architecture and core concepts
- Nodes, topics, services, and actions
- Building ROS 2 packages with Python
- Launch files and parameter management

## Learning Objectives

By the end of Week 5, students should be able to:
- Create ROS 2 packages from scratch
- Write publishers and subscribers in Python
- Implement service servers and clients
- Use launch files to start multiple nodes
- Manage parameters for different configurations
