## Capstone Project Requirements

The final capstone project requires a simulated humanoid robot that:

1. Receives a voice command (Whisper)
2. Plans a sequence of actions (LLM-based cognitive planner)
3. Navigates to a target location (Nav2)
4. Identifies and locates objects (Computer Vision)
5. Manipulates objects using robotic arms

## Evaluation Criteria

- **Functionality**: Does the system work reliably?
- **Integration**: Are all modules connected properly?
- **Code Quality**: Is the code well-structured and documented?
- **Innovation**: Does the project go beyond baseline requirements?
